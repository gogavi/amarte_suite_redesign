## 2. Universo narrativo: "El Planeta Romántico"

El gran acierto de la marca es que **el logo no es decorativo: es narrativo.** Todo el sistema visual cuenta una historia espacial-romántica que debe gobernar cada pieza.

| Elemento del logo | Significado de marca | Cómo se explota |
|---|---|---|
| 🚀 **Cupidos astronautas** (casco espacial) | Viajeros del amor; mensajeros que cruzan el espacio para unir parejas | Mascotas/íconos, stickers, ilustración editorial, loaders de app |
| 🪐 **Corazón-planeta** (cráteres tipo luna) | El destino: el "Planeta Amarte" | Fondo de portadas, ícono de app, mapa de fidelización "Pasaporte" |
| ➰ **Swash infinito** bajo el wordmark | Amor infinito / órbita | Divisores gráficos, subrayados, transiciones |
| 💖 **M en forma de corazón** | El amor en el centro del nombre | Favicon, monograma, marca de agua |

**Vocabulario visual permitido (paleta narrativa):** planetas, órbitas, estrellas, constelaciones, cráteres, cascos espaciales, naves, pasaportes, sellos, alas, cupidos. Coherente con campañas ya existentes ("Pasaporte A-Marte", "Viaja a otro planeta").

> 🧭 **Regla de oro narrativa:** si una pieza no podría existir "en el Planeta Romántico", está fuera de marca.

## 3. Logotipo

### 3.1 Versiones oficiales (del manual)

| Versión | Uso recomendado |
|---|---|
| **Positivo a color** | Uso principal sobre fondos blancos/claros. |
| **Negativo a color** | Sobre fondos negros/oscuros y fotografía. |
| **Positivo 2 colores** (magenta + negro) | Impresión económica, serigrafía. |
| **Negativo 2 colores** | Fondos oscuros con limitación de tintas. |
| **Positivo B/N** | Documentos, fax, grabado, sellos. |
| **Negativo B/N** | Fondos oscuros monocromáticos. |

### 3.2 Estructura

- **Imagotipo** = símbolo (cupidos + corazón-planeta) **+** wordmark ("AMARTE suite").
- El wordmark **puede** usarse solo en espacios muy reducidos (favicon, app icon), priorizando la **M-corazón**.
- "suite" siempre en minúscula, fuente manuscrita ligera, alineada a la derecha bajo "AMARTE".

## 4. Área de protección y tamaños mínimos

- **Área de protección:** se toma la **altura de la letra "A"** del wordmark y se aplica como margen libre en los 4 lados. Ningún texto, imagen o borde puede invadir esa zona.
- **Tamaño mínimo impreso:** **2 cm de ancho** (imagotipo completo legible).
- **Tamaño mínimo digital:** 120 px de ancho para el imagotipo; por debajo usar solo el monograma M-corazón.

```
   ┌──────────[ A ]──────────┐
   │                         │
 [ A ]   🪐 IMAGOTIPO    [ A ]   ← margen = altura de "A"
   │                         │
   └──────────[ A ]──────────┘
```

## 5. Usos correctos e incorrectos

| ✅ Correcto | ❌ Incorrecto |
|---|---|
| Mantener proporciones originales | Estirar, comprimir o rotar |
| Usar versiones provistas | Recolorear el símbolo a antojo |
| Respetar área de protección | Pegar texto sobre el logo |

| Sobre fondos con contraste suficiente | Sobre fondos rosados/saturados que lo apaguen |

## 7. Tipografía

### 7.1 Sistema oficial (manual)

| Jerarquía | Fuente | Carácter | Google Fonts |
|---|---|---|---|
| **Logotipo** | **Bellerose** | Art déco, geométrica, exclusiva del logo | ❌ (licencia/comercial) |
| **Títulos** | **Fjalla One** | Condensada, fuerte, impactante | ✅ |
| **Subtítulos** | **Courgette** | Manuscrita, romántica, cálida | ✅ |
| **Cuerpo de texto** | **Open Sans** | Neutra, altísima legibilidad | ✅ |

### 7.2 Reglas de uso

- **Bellerose se reserva solo al logo.** Nunca para textos.
- **Fjalla One** en mayúsculas para titulares cortos y de impacto.
- **Courgette** en dosis pequeñas (frases románticas, "suite", citas). Nunca en párrafos largos ni en mayúsculas.
- **Open Sans** para todo cuerpo, formularios, precios, legales.

### 7.3 Escala tipográfica sugerida (web/app)

| Token | px | Fuente | Uso |
|---|---|---|---|
| `display` | 48 | Fjalla One | Hero |
| `h1` | 32 | Fjalla One | Títulos sección |
| `h2` | 24 | Fjalla One | Subsección |
| `accent` | 20 | Courgette | Frases románticas |
| `body` | 16 | Open Sans | Texto base |
| `caption` | 13 | Open Sans | Notas, legales |

## 8. Tono de voz y copy

| Dimensión | Lineamiento |
|---|---|
| **Voz** | Cómplice, sensual, con humor pícaro y ternura. Tutea ("tú", "atrévete", "déjate llevar"). |
| **Emojis** | Sí, con criterio: 🔥💖🪐🚀😏✨🛀. Refuerzan el universo, no lo saturan. |
| **Léxico marca** | "Planeta romántico", "viaja", "aventura", "pasión", "escápate", "de otro planeta". |
| **Evitar** | Vulgaridad explícita gratuita, tecnicismos fríos, promesas que el servicio no cumple. |
| **CTA tipo** | *¡Reserva ahora! · Viaja a otro planeta · Pide tu pasaporte · Tu aventura a un clic.* |

> 🎯 **Calibración:** sensual y atrevido, **nunca** soez. La marca seduce, no incomoda.

| Activo | Plataforma | URL |

| Tienda | Shopify | hotelamartesuite.store / amartesuite.store |

### 9.2 Principios de UI

- **Fondo oscuro como base** (negro/carbón) → realza el magenta y evoca el espacio.
- **Magenta solo para acción** (CTA, links, énfasis). No usar como fondo de grandes áreas: vibra y cansa.
- **Rosa cuarzo** para superficies suaves y estados hover/seleccionado.
- **Aire generoso** y fotografía de suites a sangre completa.
- **Botones:** esquinas redondeadas (8–12px), magenta sólido, texto blanco Open Sans bold.
- **Iconografía:** línea simple, temática espacial (planeta, cohete, sello, corazón).

## 10. Diagnóstico de coherencia y propuestas

Resultado de auditar manual + amartesuite.com + hotelamartesuite.store + IG.

### 🔴 Hallazgos

| # | Hallazgo | Riesgo |
|---|---|---|
| 1 | **3 magentas distintos** conviviendo: `#C3007A` (manual), `#CC007A` y `#FF0099` (tienda) | Marca se ve "diferente" en cada canal |
| 2 | **Cian de portadas sin ficha** técnica oficial | Diseñadores no pueden replicarlo |
| 3 | **Tipografía web = Jost**, no las 4 oficiales | Quiebre de identidad tipográfica |
| 4 | **Bellerose** sin respaldo de licencia documentado | Riesgo legal/operativo |
| 5 | Manual sin **modo oscuro**, sin **tokens**, sin **accesibilidad** | No sirve "tal cual" para apps |

### 🟢 Propuestas (con fundamento)

**P1 — Unificar el magenta.** Adoptar un único magenta de marca. Recomiendo conservar el oficial **`#C3007A`** para impresión y definir **`#E6007E`** como "Magenta Digital" (más vivo en pantalla, a medio camino entre el oficial y el neón de Shopify, sin llegar al `#FF0099` que es estridente y falla en contraste). Migrar la tienda de `#FF0099` → `#E6007E`.

**P2 — Oficializar el cian como color de acento.** El cian ya vive en las portadas; en vez de eliminarlo, **promoverlo a "Acento Orbital"** con ficha: propongo **`#19A6E0`**. Da frescura "espacial" y contrasta con el magenta (complementario frío). Uso limitado: detalles, ilustración, estados informativos.

**P3 — Resolver tipografía.** Dos caminos:
- **Conservador (recomendado):** volver a las oficiales en web. Fjalla One + Open Sans ya están en Google Fonts → fácil. Courgette para acentos.
- **Evolución:** si se quiere un look más premium/moderno, reemplazar Open Sans por **Jost** (ya en uso) como cuerpo y **mantener Fjalla One** en títulos. Jost es geométrica art-déco, **coherente con Bellerose**. → Esta es una mejora válida que vale la pena adoptar y documentar.

**P4 — Sustituto seguro de Bellerose.** Como el logo ya está vectorizado, Bellerose solo se necesita si se rehace el wordmark. Mantener el logo como **vector cerrado** y no depender de la fuente. Para piezas que imiten el estilo: **Poiret One** o **Megrim** (Google Fonts, art-déco) como sustitutos legales.

**P5 — Paleta extendida funcional (nueva).** Agregar colores de sistema que el manual no tiene pero toda app necesita:

| Token | HEX | Uso |
|---|---|---|
| Éxito | `#2EB872` | Confirmaciones, reserva ok |
| Alerta | `#F5A623` | Avisos |
| Error | `#E5384B` | Errores (≠ magenta de marca) |
| Fondo oscuro | `#141414` | Base modo oscuro |
| Superficie | `#1F1F22` | Tarjetas modo oscuro |

> ✅ Todas estas propuestas **respetan el ADN** (magenta + espacio + romance) y solo **corrigen inconsistencias y llenan vacíos**, sin reinventar la marca.

### 🗺️ Escenarios

| Escenario | Esfuerzo | Resultado |
|---|---|---|
| **A — Mínimo** | Bajo | Solo unificar magenta (P1). Corrige el 80% del ruido visual. |
| **B — Recomendado** | Medio | P1+P2+P3 conservador+P5. Marca consistente y lista para apps. |
| **C — Evolución** | Alto | B + Jost como cuerpo + sistema de ilustración espacial propio. Marca premium 2026. |

## ✅ Checklist de aprobación de pieza

- [ ] ¿Usa el magenta de marca exacto (no aproximado)?
- [ ] ¿Respeta el área de protección del logo?
- [ ] ¿La tipografía corresponde a su jerarquía?
- [ ] ¿El tono de copy es sensual pero no soez?
- [ ] ¿Encaja en el universo "Planeta Romántico"?
- [ ] ¿Contraste suficiente / legible (incl. modo oscuro)?
- [ ] ¿Versión correcta del logo para ese fondo?
